# Final Assignment requirements

<br />

1. Student presentation (HTML + CSS Assignment)
2. Interactive buttons (HTML event listeners)
3. Three.js scene + photogrammetry mesh
4. Custom map
5. Deploy externally (GitHub Pages)

and the logic in **ONE or more JavaScript** file.
